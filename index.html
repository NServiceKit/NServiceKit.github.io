    Forever free, open source web services framework for .NET

    Project maintained by NServiceKit Hosted on GitHub Pages — Theme by mattgraham

    NServiceKit is a forever free, open source fork of ServiceStack V3.

    The goal of this fork is to provide a high quality, well-tested, concise framework for building web services for .NET. We have a special focus on web services on the server side (both Microsoft and Mono) and will strive to be the best framework that works well with others.

    Near Term Project Goals

    Add and enhance community requested features

    • Add features that the community wanted in ServiceStack V3
    • Enhance limited features/functionality
    • Work better with other projects

    Clean up code

    • Remove deprecated obsolete features and general cruft
    • Remove unnecessary platform targets
    • Document
    • Unit Test

    Support Mono

    • Build on Travis with Unit Tests
    • Take requests for fixes on Mono
