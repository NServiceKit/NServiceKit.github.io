NServiceKit

Forever free, open source REST services framework for .NET

NServiceKit is a forever free, open source fork of ServiceStack V3.

Near Term Project Goals

Add and enhance community requested features

  • Add features that the community wanted in ServiceStack V3
  • Enhance limited features/functionality

Clean up code

  • Remove deprecated obsolete features
  • Remove unnecessary platform targets
  • Document
  • Unit Test

Support Mono

  • Build on Travis with Unit Tests
  • Take requests for fixes on Mono

Please create issues for us if you have a feature/bug fix/enhancement. We welcome pull requests as well!
